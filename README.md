# Eduarda Vilar Nail Designer - Site Oficial

Site moderno e responsivo para Eduarda Vilar Nail Designer, especialista em Banho de Gel e Alongamento.

## 🌐 Como Colocar o Site no Ar (GitHub Pages)

### Passo 1: Criar Conta no GitHub
1. Acesse [github.com](https://github.com)
2. Crie uma conta gratuita (se ainda não tiver)
3. Verifique seu e-mail

### Passo 2: Criar Novo Repositório
1. Clique no botão **"+" no canto superior direito**
2. Selecione **"New repository"**
3. Nome do repositório: `eduarda-vilar-site`
4. Marque **"Public"**
5. Clique em **"Create repository"**

### Passo 3: Fazer Upload dos Arquivos
1. Na página do repositório, clique em **"Add file"** → **"Upload files"**
2. Arraste todos os arquivos para a área de upload:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `logo.png/` (pasta completa)
   - `certificado.jpg/` (pasta completa)
   - `hero-bg.jpg/` (pasta completa)
   - `unhas-vermelhas.jpg/` (pasta completa)
   - `alongamento.jpg/` (pasta completa)
3. Role até o final e clique em **"Commit changes"**

### Passo 4: Ativar GitHub Pages
1. No repositório, clique em **"Settings"**
2. No menu lateral, clique em **"Pages"**
3. Em "Build and deployment", selecione:
   - **Source**: Deploy from a branch
   - **Branch**: main
   - **Folder**: / (root)
4. Clique em **"Save"**

### Passo 5: Aguardar Deploy
1. Aguarde alguns minutos
2. Volte à página de Settings → Pages
3. Seu site estará disponível no URL mostrado (ex: `https://seu-usuario.github.io/eduarda-vilar-site/`)
